Bijoux a Ouro
É um site de compra virtual de bijuteriais
para entrar nele, abre a pasta de arquivos e clicque com o botão direito no arquivo so site, que é E-comerce.zip, e clique em extrair tudo
Ao extrair, de dois cliques no arquivo extraido e entre no aruivo chamado commerce.html
ao entrar nele, tem os botões podendo ir nas categorias e ao rolar para baixo, tem a parte das promções da semana
podendo adicionar o produto ao carinho, só clicando em adicionar produto ao carrinho
no início da página principal, tem os botões com as categorias, feminino, masculino e infantil, para ir na categoria, é só clicar no botão
ao clicar em uma catgoria, você será direcionado para outra página, com outros produtos, para adicioar o produto ao carrinho, é só clicar no botão "adicionar ao carrinho" que o item irá para lá
O site tem uma area de busca, queé so clicar na barra de pesquisa e pesquisar o item que você quer, para encontrar ele 
toda categoria tem um botão na tela escrito " Voltar para o início" que irá voltar para a página principal do site
ao clicar no botão "ver carrinho" que estará em todas as páginas do site, você sera direcionado para a página de pagamento, mostrando o total de itens no carrinho e o valor total. você pode retirar um item do carrinho, clicando no botão "Remover 1" assim removendo 1 iem do carrinho, se o item tiver ais que uma unidade, o botão irá remover somente 1 e não o produto inteiro
Agora é só clicar no botão "Finalizar pagamento" que você irá para outra página, onde pede o seu nome, telefone, CEP, endereço e etc. você terá que preencher tudo os itens com '*" significa que é obrigtório prenche, os que não estiverem com o "*" significa que é opcional preencher ele ou não
ao preencher tudo, ao rolar a tela, terá a opção de esclher a forma de pagamento entre, pix, cartão de credito e débito, o esclher a opção do cartão, aparecerá um campo para preenchr pedindo o número do cartão, a data de validade e cvv
ao preencher isso tudo, clique em "Finalixar pedido" que motra que o pedido foi feito
